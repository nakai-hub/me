# 最先端歯科クリニック（架空サイト）

転職用の学習成果物として作成した、HTML/CSS と jQuery を使った 1〜3 ページのコーポレートサイトです。

## このサイトの目的

- 転職用ポートフォリオとして「自分で手を入れた部分」を説明できるようにする
- HTML/CSS（＋jQuery）の基礎で、実務に近い構成のサイトを 1 本完成させる

## ページ構成

| ファイル       | 内容                         |
|----------------|------------------------------|
| `index.html`   | トップ（ヒーロー・特徴・診療時間・地図） |
| `about.html`   | 医院について（理念・導入技術）        |
| `contact.html` | お問い合わせ（連絡先・地図）          |

## 使った技術

- **HTML5** … 意味のあるタグ（`header`, `nav`, `main`, `section`, `footer`）で構造化
- **CSS3** … Flexbox でレイアウト、`position: fixed` でヘッダー固定、メディアクエリでスマホ対応
- **jQuery 3.7** … ハンバーガーメニューの開閉（1 機能のみ）
- **Google Maps Embed** … フッター直上のアクセス欄に iframe で地図を表示

## 自分で特に手を入れたところ

1. **ヘッダーを `position: fixed` にした理由**  
   スクロールしてもナビが常に見えるようにし、どのページからでも遷移しやすくするため。

2. **地図ブロックを Flexbox で「左：連絡先 / 右：地図」にした理由**  
   PC では情報と地図を同時に確認でき、スマホでは `@media (max-width: 768px)` で縦並びにし、読みやすくするため。

3. **jQuery でハンバーガーメニューを開閉する理由**  
   スマホ表示ではナビを隠し、ボタン 1 つで開閉できるようにして、画面を広く使うため。

## 面接で話すポイント（説明できるポイント 3 つ）

1. **「ヘッダーを Flexbox で横並びにした理由」**  
   ロゴ（左）とナビ（右）を `justify-content: space-between` で両端に配置し、余白を均等にしている。

2. **「スマホ用にメディアクエリを 1 つ入れた理由」**  
   768px 以下でヘッダー内のナビを縦並びにし、ハンバーガーボタンで開閉するようにして、小さい画面でも使いやすくしている。

3. **「jQuery でメニューを開閉するようにした理由」**  
   クリックで `is-open` クラスを付け外しし、CSS の `transform` と `visibility` でスムーズに表示/非表示を切り替えている。

## ローカルで確認する方法

1. このリポジトリをクローンする
2. `index.html` をブラウザで開く（または簡易サーバーで `python -m http.server` など）
3. 「ホーム」「医院について」「お問い合わせ」のリンクでページ遷移を確認

## 注意

- 住所・電話番号・メールアドレスは架空のものです
- 地図は東京都心（東京駅周辺）の一般的な位置を表示しており、実在の医院とは無関係です
